## Installation
- Step 1: Clone repository
```
git clone https://github.com/katynguyen96/javascript-training.git
```
- Step 2: Move to folder practice1
```
cd practice1
```
- Step 3: Start JSON server
```
npm run server
```
- Step 4: Run project
```
npm start
```
